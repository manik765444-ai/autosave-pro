# Autosave-Pro: Backup Automation Made Easy
==============================================

## Description
Autosave-Pro is a Python script designed to seamlessly back up your files, ensuring your data is secure and easily recoverable. With Autosave-Pro, you can rest assured that your important files are safe from unexpected losses or system failures.

## Features
* **Automatic Backup**: Schedule backups to run at regular intervals, so you don't have to worry about losing your data.
* **Customizable**: Choose which files and folders you want to back up, and where you want to store them.
* **Secure**: Use encryption to protect your backed-up data from unauthorized access.
* **Notifications**: Receive alerts when backups are completed or if any issues arise during the process.

## Installation and Usage
To get started with Autosave-Pro, follow these simple steps:
1. **Install dependencies**: Run `pip install -r requirements.txt` in your terminal to install the necessary dependencies.
2. **Run the script**: Execute `python main.py` to start the backup process.
